# Smart-Web-Based-Calculator-With-Memory-Functions

## 💡 Web-Based Scientific Calculator Using HTML, CSS, and JavaScript
### 🔹 Project Overview
A fully functional web-based scientific calculator designed to perform arithmetic, memory, and advanced mathematical operations. Built using HTML for structure, CSS for styling, and JavaScript for logic, it demonstrates the integration of front-end technologies to create an interactive, responsive, and user-friendly application.

### 🔹 System Architecture
| Layer |	Technology |	Role |
|-------|------------|-------|
| Interface Layer	| HTML | 	Defines calculator layout and buttons |
| Presentation Layer |	CSS |	Styles the UI with grid layout, hover effects, and responsive design |
| Logic Layer |	JavaScript |	Handles arithmetic, memory, and advanced operations |
| Execution Environment |	Browser	| Runs the calculator directly without external dependencies |

### 🔹 Key Functionalities
* Perform basic arithmetic: +, −, ×, ÷

* Advanced operations: square, square root, inverse, sign toggle

* Memory functions: MC, MR, M+, M−, MS

* Error handling for invalid inputs

* Responsive design with CSS grid and hover effects

### 🔹 Key Insights
* Event-driven logic: Each button triggers a JavaScript function dynamically.

* DOM manipulation: Real-time updates to the display using document.getElementById().

* Scalable design: Easy to extend with scientific or trigonometric functions.

* Separation of concerns: HTML for structure, CSS for design, JS for logic ensures maintainability.

### 🔹 How to Run
* Save files as index.html, style.css, and script.js.

* Link CSS and JS in the HTML file.

* Open index.html in any modern browser.

* Use the calculator directly — no installation required.

### 🔹 Project Objective
* Develop a responsive, interactive calculator using HTML, CSS, and JavaScript.

* Provide support for both basic arithmetic and advanced mathematical operations.

* Demonstrate event-driven programming and DOM manipulation principles.

* Ensure usability, scalability, and clean UI design for enhanced user experience.
